Esta la prueba técnica que me fue asignada para la vacante en ZIURSOFARE.

-------------------------------------------------------------------------------------

En VS 2017 con ADO.NET Entity Framework 6.0, lenguaje c#. 
 
Crear tabla en sql server 2012
 
Tabla Cliente : IdCliente (primary key), Nombres, Apellidos, Dirección
 
Crear una pantalla en windows forms donde tenga las siguientes opciones nuevo, 
actualizar, eliminar, con los campos de la tabla de clientes.
 
--------------------------------------------------------------------------------------


1. Para ejecutar la aplicación windows form descarge el project desde 
   https://github.com/Centallas/JobsTest

2. En la base de datos de su preferencia, ejecute el script CreateCustomerTable.sql para crear la tabla

3. En el archivo App.Config modifique la cadena de conexión a la bd Local de su preferencia.
   data source=(local)\(Instancia bd);initial catalog=(Nombre bd a usar);integrated security=True

4. Con visual Studio 2007 abra PRUEBA.sin y corra el proyecto.
